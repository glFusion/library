Library Plugin

The Library Plugin allows you to create custom paypal fields for your user accounts.

This version requires glFusion 1.2.0 or higher.

Library Plugin Configuration Options

General Settings
Option Description
Show on user menu? Select "Yes" to have a menu option for the Library shown on the plugin menu. Selecting "No" will prevent the automatic creation of a user menu option.
Days to keep items on hold for waitlisted users Enter the number of days to keep items "on hold" for a waiting user AFTER the item has been checked in by the previous user.
Items to show per page Enter the number of items to show per page in the item catalog.
Enable Comments? Select whether comments will be enabled for Library items.
Enable Ratings? Select whether user ratings will be allowed for Library items.
Display glFusion blocks Select which blocks will be displayed with the product catalog.
Maximum number of days an item may be checked out Enter the default maximum number of days that an item may be checked out. This can be overridden by the Librarian at checkout time.
Images and Paths
Max number of item images Enter the maximum number of items that can be associated with an item.
Max. Product Image Size Enter the maximum file size for uploaded images, in bytes.
Max Thumbnail Dimension (px) Enter the thumbnail size, in pixels. The uploaded images will be resized so that the maximum dimension fits within this value while maintaining the aspect ratio.
Max Image Width (px) Enter the maximum image width, in pixels, for uploaded images. Images will be resized so that the width fits within this value while maintaining the aspect ratio.
Max Image Height (px) Enter the maximum image height, in pixels, for uploaded images. Images will be resized so that the height fits within this value while maintaining the aspect ratio.
Path to Images Enter the full path to where uploaded images are to be stored. Your webserver must be able to read and create files in this directory.

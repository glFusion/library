PayPal Plugin

PayPal Plugin - Product Information Form Help

Main Product Settings
Option Description
Product Name This is the short product name or item number. This will be submitted to Paypal and shown on the customer's receipt.
Category Select the category for this product.
Short Description Enter a short description for this product. This will be displayed under the product name in the product lists.
Description Enter the full description for this product. This will be displayed on the product detail screen.
Keywords Enter any additional search terms for this product. The name, short description and full description are already searched, so this field should contain words that do not appear in any of the other fields.

These words will not appear anywhere with the product.
Price Enter the unit price for this product. Only numbers should be entered here- no commas or currency symbols.
Images You may upload one or more images, up to the maximum number and size set in the Paypal configuration. You may also delete existing images by clicking the "Delete" link next to the image thumbnail.

These images will be shown as thumbnails, and can be expanded using the Slimbox viewer. You can also use the advanced editor to incorporate images directly into your product description.

You might want to have at least one image to serve as a product thumbnail.
Expiration Time Enter the number of days after which a purchase will expire. This is normally used for downloadable products to allow a buyer to download the item only within a certain period.
Enabled? This checkbox allows you to easily enable or disable products. A product that is disabled will not appear in the catalog or search results.
Featured? Select whether to feature this product. Featured products will appear in their own side block, if that block is enabled.
Taxable? Select whether this product is subject to sales tax. The default is "true", which means that sales tax will be applied according to your PayPal account configuration. If you have not configured PayPal to automatically apply sales tax, then this checkbox has no effect.

The primary purpose for this checkbox is to make certain items non-taxable.
Buttons Select the button types that will be available for this product. To improve performance, especially if encrypted buttons are used, the buttons are created and stored with the product record each time is it updated.
Comments Select the comment option for this specific product. Comments will only be shown at all if the global configuration allows it; this field allows special handling per product.

Yes
Comments will be shown and users can add comments.
No
No comments will be shown.
Closed
Comments will be shown, but no additional comments will be accepted.
Delivery Information
Option Description
Downloadable? Select the type of product being sold. The current options are
  • Downloadable : A downloadable product such as a piece of software.
  • Physical : A physical product that will be shipped to the customer
Select File
(Download Only)		 Select an existing file to be downloaded.
Upload a new file
(Download Only)		 Select and upload a new file for a downloadable product. In the event that you select an existing file and upload a new one, the new file will take precedence.
Weight
(Physical only)		 This field is used by PayPal to determine shipping charges, if you've set up a shipping schedule in your PayPal account setup. Enter the weight, using the unit of measure that you use in Paypal. If the weight is set to zero, then no shipping charge will be calculated.
